Rooms to roll dice in is an online dice roller, which allows you to share your dice rolls with a group of people. It is intended to be used for tabeletop role-playing games like Call of Cthulhu or Dungeons and Dragons.

Create a Room - no account needed

Simply click on one of the buttons below to create a new room. There is no need to create an account.

How does it work?

  • Once you have created a room you can change the name and color of that room.
  • In order to share a room with other people you only need to send the URL of your room with them.
  • Dice to roll dice in remembers you by setting a cookie. If you want to change the name or color of a room, which you have created, you need to visit the room with the same browser (and device).
  • Rooms are automatically deleted, if there is no activity for a week.
  • As long as you are using the same browser and did not delete your cookies you can list your rooms using the button View existing rooms on the top right. There you also have the option to delete rooms, that were created by you.
